# Courier-Shipment-db
🚚 Courier Shipment Tracking DB

📌 Project Description

A database system designed to manage courier shipments, tracking, customers, and delivery operations.

🧠 Concepts Used

- ER Modeling
- Normalization (3NF)
- Primary & Foreign Keys
- Joins & Aggregations
- Triggers & Stored Procedures

🛠 Tech Stack

- MySQL
- SQL

📁 Folder Structure

- "/sql" → Database scripts
- "/docs" → ER diagram & report

🚀 Features

- Shipment booking
- Real-time tracking updates
- Payment handling
- Delivery agent assignment
